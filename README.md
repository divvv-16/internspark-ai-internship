# Internspark AI Internship

## About
Machine Learning internship tasks completed using Python and Google Colab.

## Tools Used
- Python 3
- Google Colab (no installation needed)
- pandas, numpy, scikit-learn, matplotlib, seaborn

## How to Run
1. Click on any notebook link below
2. Click "Open in Colab" at the top
3. Click Runtime → Run All

## Task 1 - Classification Model
- Dataset: Titanic (891 passengers)
- Goal: Predict passenger survival
- Models: Logistic Regression vs Random Forest
- Best Model: Random Forest (82.68% accuracy)

## Package Versions
- pandas: 2.x
